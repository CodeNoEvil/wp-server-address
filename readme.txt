== Description ==

WP Server Address displays the local IP address of the server in the admin toolbar. As a submenu of the IP address is the hostname, database name, and database username as well.

== Installation ==

1. Upload the `wp-server-address` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
